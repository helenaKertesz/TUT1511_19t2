Truncation of ints

Since ints cannot store decimal points, if you do maths with them any decimal points will be truncated (chopped off). For example, someone who is 177cm is 5 foot 10 inches, or 5.83 feet. However, since we are using ints the code above would truncate this to 5 feet.


while() loops

A loops lets you do the same thing multiple times until a condition becomes false, the syntax is: 
while(/*some boolean statement*/){
    // everything inside these curly braces will run
    // over and over until the statement is false
}
Make sure that whatever is happening inside the loop will eventually cause the condition to become false or it will run forever (an infinite loop).
