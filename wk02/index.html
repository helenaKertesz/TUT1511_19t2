if statements

if(/*some boolean statement*/){
    // everything inside these curly braces will only
    // be run if the statement is evaluated as true
}

You can also chain many if statements together using else if(). The actions inside will be mutually exclusive, once a condition is found to be true it will execute everything inside that set of curly braces and then skip the rest of the chain. 

if(...){
    // since we have another set of curly braces
    // everything inside here is indented once more
} else if(...){
    // you can have as many else if's as you like
    // the "else" chains it on to the if before
} else if(...){
    ...
} else {
    // you should always end on an "else" statement
}

// Author: Wed09Sitar and Wed13Kora
// Date created: 11-06-2019
// Reads in a mark and then prints pass or fail

#include <stdio.h>

int main(void) {
    int mark;
    printf("Please enter your mark: ");
    scanf("%d", &mark);

    if( mark < 0 || mark > 100 ){
        printf("ERROR\n");
    } else if( mark >= 50 ){ 
        // we don't need to check > 0 because that was done in the last if
        // we only get to this part if the first if condition was false
        printf("PASS\n");
    } else {
        // don't need to check anything because the last possible case is >= 0 and < 50
        printf("FAIL\n");
    }
    
    return 0;
}

More I/O

Here the "formatted" comes into printf() and scanf(). Within the quotation marks you can put format specifiers, and then the variable/s you want to print out / read in go afterwards.

Format specifiers start with a %. The ones you will see the most are %d which is an integer, and %lf which is a double (which is a "long float", a float being a number with decimal points). You can look at the manual page for scanf (man scanf) to see some more format specifiers.

Note that in scanf() you need to put an ampersand & before the name of the variable you are reading into.

// A very simple C program
// Given two edges, print the area of the rectangle

#include <stdio.h>

int main(void) {

    printf("Please enter the width and length of the rectangle: ");
    scanf("%d %d", &width, &length ); // assuming the user inputs space separated numbers

    area = width * length; 
    // this could be done directly in the printf but is clearer if we store it in a separate variable

    printf("Area = %d\n", area);
    
    return 0;
}
